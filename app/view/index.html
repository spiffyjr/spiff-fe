Connecting to Lich

In order to connect this application requires Lich to be running in detachable client mode. This allows Lich to run in the background and Spif FE can connect to it directly.

  1. Open cmd.exe by presssing WIN+R, typing cmd.exe, and hitting enter.
  2. Change to the Lich directory by entering, cd c:\<your lich directory>
  3. Run ruby lich.rbw --login character --without-frontend --detachable-client=8000
  4. Modify the hostname/port below and press connect.
    • If you copied the command in step 3 the defaults are correct.
